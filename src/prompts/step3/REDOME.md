# ISRステップ3（検証結果）人間検品用ジャッジマニュアル

## 1. ジャッジの全体フロー
人間の運用者が行うのは、ステップ3が出力したJSONデータを確認し、AIの校閲ぶりを評価して、次の「ステップ4（Web記事化）」へ進めて良いかを決めるだけの簡単な作業です。

---

## 2. 3大チェックポイント（OK / NG 基準）

### ① `triple_check_report`（AIによる不純物報告）の確認
AIが「動画外の一般論を見つけて、どう処置したか」のログを確認します。

*   **[ AIの自己申告が `has_hallucination: true`（不純物あり）の場合 ]**
    *   `discrepancies_found`（問題の箇所）をサッと見ます。
    *   「教科書的な心理学の定義を削った」「動画で言ってないネットの補足を消した」という報告であれば、AIが完璧に仕事をしています。➔ **【一歩前進】**
*   **[ AIの自己申告が `has_hallucination: false`（不純物なし）の場合 ]**
    *   【注意】AIがハルシネーションを見落としている（騙されている）可能性があります。次の「②」の目視チェックを少し慎重に行ってください。

### ② `cleansed_context` の「非断定表現」チェック
ここが最も重要です。ここにある文章がステップ4で記事の肉付けになるため、ニュアンスの余白を確認します。

*   **[ OK ] 寄り添いの余白がある：**
    *   「〜という傾向にあります」「〜なところが多いです」「〜なのかもしれません」といった、読者に考えさせる柔らかい語り口のベースができている。
*   **[ NG ] お説教・決めつけになっている：**
    *   「〇〇です」「〇〇ですね」という強い断定表現が乱発されており、上から目線の正論に見える。

### ③ 動画の「独自の視点」が100%残っているか
一般的なまとめサイトのような「教科書的な正論」にすり替わっていないかを確認します。

*   **[ OK ] リスペクトの維持：**
    *   世間の「いじめ＝悪」という単純な話ではなく、「当事者間の認知のズレ」「世間の正義中毒・不幸の消費」という、動画独自の切り口が濁らずに残っている。
*   **[ NG ] 一般論への埋没：**
    *   「いじめ問題の解決には、第三者の介入と迅速なリスクマネジメントが不可欠です」といった、本人が動画で熱弁していない「ネットのどこかで見たような綺麗事」に着地してしまっている。

## 3. 人間が取るべき3つのアクション

検品の結果、人間は以下の3つのうちいずれかのルートを選択してシステムを動かします。

| 検品結果（実態） | 人間のジャッジ | 次のアクション |
| :--- | :--- | :--- |
| 不純物が綺麗に消え、語尾も「〜な傾向にある」と柔らかい。動画の魂が残っている。 | **【 完全勝利（GO） 】** | そのまま `cleansed_context` をコピーして、**ステップ4（Web記事化）へ投入**する。 |
| AIのクレンジング自体は完璧だが、文章の語尾にまだ少し「〜です。」という硬さが残っている。 | **【 微修正（PASS & TWEAK） 】** | 人間の手で1〜2箇所、語尾を「〜なところが多いです」等に**手動でリライトしてからステップ4へ進む。** |
| 動画独自の視点が消え、Wikipediaのような教科書的な一般論にすり替わってしまっている。 | **【 やり直し（REJECT） 】** | ステップ3、またはステップ2のプロンプトに戻り、「一般論の排除」の制約をさらに強めて**再実行する。** |

## 運用のアドバイス（人間の目だからこそ気づけること）
AIは「論理的な正論」が大好きです。そのため、油断すると「動画の泥臭い主張」を「綺麗で退屈な一般論」にアップデートしてしまいます。

人間がチェックする際は、**「綺麗にまとまっているか」ではなく、「本人のあの切実なトーン（〜なところが多いです、というニュアンス）や、独自のトゲがちゃんと残っているか」**という視点を持つこと。それだけで、記事のクオリティは100%保証されます。
